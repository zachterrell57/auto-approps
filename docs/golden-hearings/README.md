# Golden Hearings Seed Dataset

This seed dataset gives the local `npm run hearing-eval -- run --dataset golden-hearings --mode full` harness deterministic fixtures for the MVP. It models post-capture `live_asr` transcript text for watchlist and citation regression; it is not a substitute for the broader human-reviewed 20+ real-hearing dataset required before production launch.

The cases cover House and Senate proceedings, appropriations, oversight, legislative, budget, nomination, and markup-style scenarios, plus exact, alias, bill-normalized, semantic, negative-filter, non-hit, citation, and export checks.
